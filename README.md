构建一个可视化搭建平台，通过拖拽的方式构建中台，生成完整的项目工程 [wiki](https://github.com/winyh/astx/wiki)

### 安装
```
npm install 
```

### 启动

```
yarn start

```

### 打包
```
yarn build
```
